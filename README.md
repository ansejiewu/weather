# Weather Page

独立的天气 HTML 页面，参考 iOS Weather 风格。

- 数据源：[Open-Meteo](https://open-meteo.com/)（免费，无需 API Key）
- 单文件 HTML，无外部依赖（无 CDN、无框架）
- 响应式布局（手机 / 平板 / PC）
- 24 小时温度趋势（SVG 贝塞尔曲线）
- 5 天天气预报
- 动态背景（根据天气自动切换）

## 本地运行

直接双击 `index.html` 在浏览器中打开即可。

## 部署到 GitHub Pages

1. 进入 GitHub 仓库 **Settings → Pages**
2. Source 选 **Deploy from a branch**
3. Branch 选 **main**，目录选 **/(root)**
4. 保存后几分钟后即可访问

地址：`https://ansejiewu.github.io/weather/`
